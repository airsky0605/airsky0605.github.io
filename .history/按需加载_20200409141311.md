# 按需加载

### 实现按需加载的最关键一步是要先用 babel-plugin-component 这个文件

```
npm i babel-plugin-component -D

```

### 修改.babellrc文件

```

{
  "presets": [["es2015", { "modules": false }]],
  "plugins": [
    [
      "component",
      {
        "libraryName": "element-ui",
        "styleLibraryName": "theme-chalk"
      }
    ]
  ]
}

```
